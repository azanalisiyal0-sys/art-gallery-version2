# Art Gallery V2

A responsive art gallery website built using HTML, CSS and JavaScript.

## Project Overview

Art gallery V2 is a frontend project designed to display digital artworks
in a clean and interactive gallery layout. The project focueses on improving
user interface design and JavaScript interactivity.

## Features

- Responsive design for mobile and desktop
- Interactive gallery cards
- Hover lift effects on artwork cards 
- Neon button hover effects
- Image filtering using JavaScript

## Technologies Used 

- HTML
- CSS 
- JavaScript

## Purpose of the Project

This project was created to practice frontend development
and improve skills in building interactive web interfaces.

Note: This is a frontend-only project. No user data is 
collected or stored.
